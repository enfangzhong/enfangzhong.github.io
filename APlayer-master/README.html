APlayer

Wow, such a lovely HTML5 music player







Introduction

APlayer is a lovely HTML5 music player.

APlayer supports:

  • Media formats
    • MP4 H.264 (AAC or MP3)
    • WAVE PCM
    • Ogg Theora Vorbis
  • Features
    • Playlist
    • Lyrics
